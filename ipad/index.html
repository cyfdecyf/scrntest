iPad Screen Test

Test whether your iPad screen has bad pixels.

The first two steps are required to run this page in full screen mode on your iPad.

  1. Add this page to home screen
  2. Launch this page from home screen
  3. Tap anywhere to display a sinlge color
  4. Check with your eyes
  5. Tap again to display the next color

All colors displayed, will start over again.
